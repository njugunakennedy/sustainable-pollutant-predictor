Enhanced Pollutant Removal Predictor (Multi-Factor ML Model)This Streamlit application demonstrates a Polynomial Regression machine learning model used to predict Milk Fat Removal Efficiency from dairy wastewater using a biochar adsorbent. It simulates a sophisticated adsorption study by modeling the complex, non-linear effects of four critical operational factors: $\text{pH}$, Initial Milk Fat Concentration, Biochar Dosage, and Temperature.The project is inspired by research focusing on sustainable, circular economy solutions in the dairy industry, converting waste sludge into a valuable treatment resource (biochar).🚀 FeaturesMulti-Factor Prediction: Predicts Milk Fat Removal Efficiency (0-100%) based on the interactive effects of $\text{pH}$, Concentration, Dosage, and Temperature.Polynomial Regression: Utilizes a 3rd-degree Polynomial Regression model trained on synthetic data to capture complex, non-linear relationships and factor interactions.Real-time Interaction: Use sliders in the sidebar to instantly see the predicted efficiency change.Advanced Visualization: Displays the primary $\text{pH}$ effect curve, allowing users to see how fixing the other three parameters affects the overall response.Sustainability Context: Provides background on the circular economy approach and the industrial-scale application of the biochar technology.⚙️ How to Run LocallyPrerequisitesYou need Python 3.8+ installed on your system.InstallationClone the Repository (or save the code):Save the provided Python code as a file named app.py.Install Required Libraries:The application relies on streamlit, pandas, numpy, plotly, and scikit-learn. The application will automatically open in your web browser (usually at http://localhost:8501).🔬 Core Modeling & Scientific ContextData SynthesisThe data is synthetically generated to mimic expected scientific trends in adsorption studies:pH: The primary factor, modeled with a parabolic curve peaking near $\text{pH} 4$ (Chemically optimal environment).Dosage: Modeled with a positive correlation (more adsorbent means more removal sites).Concentration: Modeled with a slight negative correlation (higher initial pollutant load slightly lowers percentage efficiency due to site saturation).Temperature: Modeled with a quadratic curve peaking around $35^\circ\text{C}$, indicating the process is likely chemisorption-driven.ML ApproachA sklearn.pipeline combining PolynomialFeatures(degree=3) and LinearRegression is used. This approach generates all possible $\text{pH}^3$ terms and all interaction terms between the four features (e.g., $\text{pH}^2 \times \text{Dosage} \times \text{Temperature}$), allowing the linear model to fit a highly complex, non-linear surface.📝 Key FunctionsFunction NameDescriptionStreamlit Decoratorsynthesize_data_enhanced()Generates the multi-feature synthetic dataset with defined non-linear effects and noise.@st.cache_datatrain_prediction_model_enhanced()Trains the 3rd-degree Polynomial Regression model using all four features.@st.cache_resourcepredict_removal_enhanced()Predicts the Milk Fat Removal Efficiency for a specific combination of ($\text{pH}$, Concentration, Dosage, Temperature).None
